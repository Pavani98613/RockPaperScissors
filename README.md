## How to Run

1. Clone or download the repository.
2. Make sure the following files are present in the same folder:
   - `index.html`
   - `style.css`
   - `script.js`
3. Open the `index.html` file in your browser (Chrome, Edge, Firefox, etc.)
4. Start playing!
